# 富邦之星學習教案網頁

這是李孟珊業務經理「富邦之星」課程的互動式學習筆記頁。

## 檔案結構

```
fubon_star_learning_project/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── assets/
│   └── fubon-star-hero.png
├── .nojekyll
└── README.md
```

## 本版重點

- 依逐字稿整理成教案型筆記，不包含講師投影片或課堂截圖。
- 純 HTML/CSS/JS，無外部 CDN 依賴。
- 包含低標盤點、認同鏈檢核、邀約稿產生器、12 個月水池、社群貼文練習、挫折復原與本週行動稿。
- 表單內容會保存於瀏覽器 localStorage。
- 頂部與手機版皆含「返回目錄」按鈕：https://duduflow.github.io/learning_notes_catalog/

## 使用方式

直接開啟 `index.html` 即可瀏覽；整個資料夾可直接上傳 GitHub Pages。
